# 🗓️ Journal

A dated, freeform learning log — the literal *diary*. One file per day, named `YYYY-MM-DD.md`,
filed under the year. Low-pressure: jot what I learned, what I worked on, and what to revisit.
When an entry contains something worth keeping, I promote a clean version to [`../topics/`](../topics/).
