# AIUB

Notes from courses at **American International University-Bangladesh**, organized by semester.

## Semesters

- [2026 Spring](2026-spring/)

_(Add a folder per semester as you go: `2026-summer/`, `2026-fall/`, …)_

## Add a new course

1. Inside the semester folder, create `<course-slug>/` (e.g. `cse1101-intro-to-programming/`)
2. Copy the structure from [`2026-spring/example-course/`](2026-spring/example-course/)
3. Fill in the course `README.md` (course code, instructor, syllabus link)
