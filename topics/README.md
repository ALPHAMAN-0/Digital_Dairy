# 📚 Topics

Evergreen, distilled knowledge organized by **subject** — independent of where I learned it.
This is the library I actually search months later. When something from a course or the journal
finally *clicks*, I write a clean version here.

## Subjects

- [Programming](programming/) — languages, paradigms, data structures, algorithms
- [Computer Science](computer-science/) — theory, OS, databases, math foundations
- [Networking](networking/) — protocols, models, the web stack
- [Software Development](software-development/) — git, testing, design, tooling, practices

## How to add a topic note

1. Copy [`../_templates/topic-note.md`](../_templates/topic-note.md)
2. Name it in `kebab-case` (e.g. `binary-search.md`)
3. Drop it in the right subject folder
4. Fill the frontmatter (`tags`, `source`, `status`) and link related notes
