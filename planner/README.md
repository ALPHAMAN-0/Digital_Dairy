# 🚀 Planner

Your productivity hub. The diary captures **what you learn**; the planner drives **what you do**.

## What's here

- [🎯 Goals](goals.md) — learning goals with deadlines and progress bars
- [✅ Habits](habits/) — monthly study-streak grids
- [🗓️ Weekly reviews](weekly/) — plan the week ahead, reflect on the last one
- **Daily to-dos** live in your daily log → [journal/](../journal/) (the `Plan today` / `Done` sections)

## The rhythm

| When | Do this |
|---|---|
| **Every day** | Open today's [daily log](../journal/), fill **Plan today**, tick off **Done**, jot what you learned. |
| **Every week** | Copy [`../_templates/weekly-review.md`](../_templates/weekly-review.md) → set focus, review wins + carry-overs. |
| **Every month** | Start a new habit grid from [`../_templates/habit-month.md`](../_templates/habit-month.md). |
| **When a goal moves** | Update its progress bar in [goals.md](goals.md). |

> Everything is plain Markdown — check boxes off in any editor, on GitHub, or in Obsidian.
